# frontless-js
## Installation
```console
npx create-frontless-app my-app
cd my-app
```
## run
### Option.1: using Bun
If not installed Bun
```console
curl -fsSL https://bun.sh/install | bash
```
run
```console
bun server.ts
```

### Option.2: using Node
run
```console
npx tsx --env-file .env server.ts
```
